# Libraries

cvzone is a Python library for easy to run Image processing and AI functions.
NumPy is a Python library used for working with arrays.
math is a Python library use to perform various mathematical calculations

## Installation

Run 'pip install cvzone' to install cvzone.
Run 'pip install math' to install math.
Run 'pip install numpy' to install numpy.

```bash
pip install cvzone
pip install math
pip install numpy
```
